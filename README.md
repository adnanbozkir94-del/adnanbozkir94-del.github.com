```markdown
# 3D Araba Oyunu — mkbil (Basit Başlangıç)

Bu proje, Babylon.js kullanarak hazırlanmış küçük bir 3D araba oyunu iskeletidir. CDN üzerinden Babylon.js modülü kullanılır, bu yüzden basit bir HTTP sunucusu ile çalıştırılması yeterlidir.

Çalıştırma:
1. Proje klasörüne girin.
2. Basit bir HTTP sunucusu çalıştırın (örnekler):
   - Python 3: `python -m http.server 8000`
   - VS Code Live Server uzantısı
3. Tarayıcıda `http://localhost:8000` adresini açın.

Kontroller:
- W / ↑ : İleri
- S / ↓ : Geri
- A / ← : Sol
- D / → : Sağ

Geliştirme fikirleri:
- Gerçek araba modeli (.glb) ekle
- Fizik motoru (Cannon/Ammo) entegre et
- Kamera çeşitleri ve UI (HUD) ekle
- Yarış pisti, AI rakipler, checkpoint sistemi
```
